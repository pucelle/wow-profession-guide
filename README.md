### 关于

这是一份 2009 年制作的魔兽世界商业技能指南，最早是 TBC 版本，后来又更新到 WLK 版本。

那时我刚刚开始学习 Web 编程，这差不多是我的第一个像样的项目。

转眼怀旧服即将开放，我想这个在我的硬盘上待了十年的小项目也许还能发挥一些光和热。
